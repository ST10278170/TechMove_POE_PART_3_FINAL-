# TechMove.GLMS

Lightweight GLMS application (ASP.NET Core / Razor Pages + MVC controllers). This repository contains the web app and a tests project.

Quick start
- Restore packages: dotnet restore
- Build: dotnet build
- Run the web app: dotnet run --project TechMove.GLMS
- Run tests: dotnet test

Notes
- The tests project contains unit tests and an integration test harness. If tests fail locally, run dotnet restore then dotnet build first.
